================================================================================
DEPLOYMENT COMMANDS
================================================================================

1. Navigate to DurableBackend Directory
----------------------------------------
cd "c:\Users\maddi\OneDrive\Desktop\multi_agents\Ai_ Frontend\DurableBackend"


2. Publish to Azure (Standard)
-------------------------------
func azure functionapp publish durable-backend-fn


3. Publish to Azure (Verbose Output - Recommended)
---------------------------------------------------
func azure functionapp publish durable-backend-fn --verbose


4. Publish with Native Dependencies
------------------------------------
func azure functionapp publish durable-backend-fn --build-native-deps --verbose


================================================================================
WHAT YOU'LL SEE DURING DEPLOYMENT
================================================================================

1. Resolving runtime - Detecting .NET isolated runtime
2. Building project - Compiling your code
3. Packaging - Creating deployment package
4. Uploading - Progress bar showing upload (~18.4 MB)
5. Syncing files - Deploying to Azure
6. Health check - Verifying deployment
7. Function URLs - List of deployed functions with their endpoints

Expected Output Example:
------------------------
StartConversation - [httpTrigger]
    Invoke url: https://durable-backend-fn.azurewebsites.net/api/start/{input}


================================================================================
MONITORING & VERIFICATION COMMANDS
================================================================================

List All Deployed Functions
----------------------------
func azure functionapp list-functions durable-backend-fn


List Functions with Access Keys
--------------------------------
func azure functionapp list-functions durable-backend-fn --show-keys


Stream Live Logs from Azure
----------------------------
func azure functionapp logstream durable-backend-fn


Check Function App Status
--------------------------
az functionapp show --name durable-backend-fn --resource-group durable-rg --query "state"


Get Function App Settings
--------------------------
az functionapp config appsettings list --name durable-backend-fn --resource-group durable-rg


================================================================================
MANAGEMENT COMMANDS
================================================================================

Restart Function App
--------------------
az functionapp restart --name durable-backend-fn --resource-group durable-rg


Stop Function App
-----------------
az functionapp stop --name durable-backend-fn --resource-group durable-rg


Start Function App
------------------
az functionapp start --name durable-backend-fn --resource-group durable-rg
